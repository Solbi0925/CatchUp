## 한 줄 소개

강의계획서, LMS 공지, 과제, 시험, Google Calendar 일정을 AI가 정리해 오늘과 이번 주에 해야 할 일을 추천해주는 앱 기반 학업 일정 매니저입니다.

## 해결하려는 문제

대학생은 학업 일정과 개인 일정이 여러 채널에 흩어져 있어 중요한 마감일을 놓치거나 무엇부터 해야 할지 판단하기 어렵습니다.

## 타깃 사용자

- 여러 과목을 동시에 듣는 대학생
- LMS 공지와 강의계획서를 자주 확인해야 하는 학생
- 과제, 시험, 개인 일정을 함께 관리해야 하는 학생
- 일정 관리가 어렵거나 마감 직전에 몰아서 하는 학생

## 1차 MVP 범위

- 강의계획서, LMS 공지, 과제 명세서 등 학업 자료 업로드
- 최초 구현에서는 PDF 또는 이미지 중 하나의 파일 형식만 지원
- AI 기반 과제명, 마감일, 시험일, 제출 방식, 중요 일정 추출
- 추출된 정보 확인 및 수정
- Google Calendar 기반 일일/주간 학습 계획 생성
- 계획 생성 시 자연어 요구사항 입력
- 현재 상태 기준 앞으로 4주 일정을 참고해 이번 주 계획 업데이트
- 완료한 일 제거, 미완료 항목 재배치, 사용자 수정 일정 유지 중심의 주간 계획 업데이트
- AI 챗봇을 통한 계획 수정 요청
- 추천 계획에 대한 간단한 근거 설명
